# GitHub Student Developer Pack Benefits

The GitHub Student Developer Pack provides students with free access to professional developer tools and services. Here's a comprehensive list of all benefits:

## Core GitHub Benefits

- **GitHub Pro** - Free GitHub Pro account while you're a student
- **GitHub Copilot** - Free access to Copilot Pro (AI pair programmer)
- **GitHub Codespaces** - Free Pro level access to cloud development environments
- **GitHub Pages** - Unlimited project sites for hosting
- **GitHub Certification** - 1 free voucher for GitHub Foundations or Copilot certification exam (expires June 30, 2026)

## Cloud & Infrastructure

- **Microsoft Azure** - $100 credit + free access to 25+ cloud services (ages 18+)
